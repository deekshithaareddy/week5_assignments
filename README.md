# Week 5 Assignments


## Projects

### 1. Tailwind CSS - Image Grid

A responsive image card layout built with Tailwind CSS.

**Concepts practiced:**
- Tailwind utility classes (`flex`, `text-*`, `bg-*`, `py-*`, `my-*`)
- Responsive text color variants with breakpoints (`sm:text-blue-300`)
- Flexbox layout via Tailwind
- Custom color classes (e.g., `text-mauve-700`)



### 2. Tailwind CSS - Landing Page (`week5-tailwind-landing.html`)

A simple landing page layout with a hero section, content cards, and a footer.

**Concepts practiced:**
- Arbitrary value utilities (`p-[70px]`, `text-[15px]`, `rounded-[5%]`)
- Tailwind spacing scale (`gap-5`, `mb-5`, `mx-10`, `ml-20`)
- Styled label-button combos as call-to-action elements
- Horizontal rule and footer layout



### 3. Tailwind CSS - Themed Page (`week5-tailwind-themed.html`)

A red-themed political/brand-style page using dark color combinations.

**Concepts practiced:**
- CDN-based Tailwind (`<script src="https://cdn.tailwindcss.com">`)
- Dark background variants (`bg-red-950`)
- Width fractions (`w-1/3`)
- Nested flex containers
- Custom padding offsets (`pl-32`, `py-20`)



### 4. HTML Tables (`week5-tables.html` + `5.css`)

Three distinct table examples demonstrating different `colspan` and `rowspan` use cases.

**Tables included:**

| Table | Description |
|-------|-------------|
| Table 1 | Generic colspan/rowspan grid structure |
| Table 2 | Student grade report (Biology, Chemistry, Physics) with rowspan subjects |
| Table 3 | Styled users table with profile images, email, and phone columns |

**Concepts practiced:**
- `colspan` and `rowspan` attributes
- `<thead>` and `<tbody>` structure
- External CSS table styling (`border-collapse`, `padding`, custom colors)
- Embedding images inside table cells



### 5. Flexbox - Fruit Cards (`flex1.html` + `flex1.css`)

A fruit showcase page using circular images and colored name badges.

**Concepts practiced:**
- `display: flex` with `justify-content: space-around`
- `border-radius: 100%` for circular images
- `gap` for spacing between flex children
- `text-align: center` on flex sections
- Per-class background colors for badges



### 6. Flexbox - Blog Layout (`flex2.html` + `flex2.css`)

A full-page blog layout with a navbar, hero section, article cards, and a footer.

**Concepts practiced:**
- Flex navbar with `align-items: center` and `flex-wrap`
- Card-based article grid (`justify-content: space-around`)
- Multi-section footer (`justify-content: space-between`)
- Label + input button pattern for "Read More" CTAs
- Global reset with `* { margin: 0; padding: 0; box-sizing: border-box; }`



## Key Concepts Covered

- **HTML Tables** — `colspan`, `rowspan`, `<thead>`, `<tbody>`
- **CSS Flexbox** — `display: flex`, `justify-content`, `align-items`, `gap`, `flex-wrap`
- **Tailwind CSS** — utility classes, arbitrary values, responsive prefixes, CDN vs build setup
- **Semantic HTML** — proper use of `<nav>`, `<main>`, `<footer>`, `<section>`
- **CSS Reset** — universal box model normalization
